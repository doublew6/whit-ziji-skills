# 每日优先级筛选

## 目标

每天从 Reminders 未完成待办中，根据北极星目标动态挑选最重要的1件事和3件事。

## 使用时机

- 用户问"今天做什么"
- 用户问"帮我安排今天"
- 用户问"每天任务建议"

## 输入

- Reminders "待办事项" 列表的未完成任务
- 北极星目标（从 MEMORY.md 或其他配置文件动态读取）

## 工作流程

### Step 1: 读取 Reminders 未完成待办

使用 AppleScript 读取：
```
tell application "Reminders" to tell list "待办事项" to get name of reminders whose completed is false
```

### Step 2: 读取北极星目标

读取 workspace 下的以下文件获取当前优先级：
- MEMORY.md
- SOUL.md
- 或其他定义目标的文件

根据文件内容动态判断当前最高的优先级领域。

### Step 3: 筛选任务

- **第1优先**：与当前北极星直接相关的任务
- **第2-3优先**：为北极星服务的次要任务或近期必须完成的事项
- **排除**：与北极星无关的杂事（除非紧急）

### Step 4: 输出

```
🎯 今天最重要的1件事
[具体任务]

📝 今日3件事
1. [任务1]
2. [任务2]
3. [任务3]
```

## 注意事项

- 只读取"待办事项"列表
- 忽略已完成的（completed is false）
- 不要写死具体领域，根据北极星动态判断
- 如果无法判断北极星，询问用户当前最高优先级是什么
